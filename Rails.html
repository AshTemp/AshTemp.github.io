How to use Rails on to Cloud9
    1. Clone your GitHub repo
    2. Go over to Cloud9 and click Create new workspace
      3. Paste the link from your GitHub account into the box, create a workspace name and description and select RUBY as your template.
      4. Go down to your bash terminal and enter the following commands:
      gem install rails
      bundle install
      rails server -b $IP -p $PORT
      createdb operationcode_development
